# Mehrgiah Shop Clone

A responsive frontend implementation inspired by the Mehrgiah website, built for learning and practicing modern React development.

> **Note:** This project is a personal educational practice and is not affiliated with the original Mehrgiah website.

## Features

- Responsive design (Mobile & Desktop)
- Product listing
- Product details page
- Product search
- Shopping cart
- Add / Remove products
- Increase / Decrease quantity
- Cart persistence using Local Storage
- Sidebar navigation
- Dynamic routing with React Router
- API data fetching with Axios
- Skeleton loading while fetching data

## Built With

- React
- React Router
- Redux Toolkit
- Axios
- Tailwind CSS
- React Icons

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
